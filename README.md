# Indian Software Industry Salary Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on an Indian software industry salary dataset to understand salary trends across job roles, locations, and company ratings.

## Analysis Performed
- Data cleaning and preprocessing
- Salary distribution analysis
- Job role-wise salary comparison
- Location-wise salary analysis
- Mean vs median salary comparison
- Rating vs salary correlation analysis

## Key Insights
- Salary data is right-skewed due to high-paying roles.
- Senior-level roles earn significantly higher salaries.
- Median salary gives a more realistic estimate than mean.
- Metro cities dominate higher salary brackets.
- Company rating shows weak correlation with salary.

## Conclusion
This project demonstrates a complete data analytics workflow including data cleaning, exploratory analysis, visualization, and insight generation.
